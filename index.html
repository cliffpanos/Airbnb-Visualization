Airbnb Visualization
Created using d3

This information visualization allows users to explore average listing prices in specific neighborhoods as well as filter locations by room type, average rating, and number of occupants. The vis also allows hosts to observe which listings are similar to theirs in their area by filtering using the aforementioned characteristics.
