Barcelona Visualization
Created using d3

This information visualization allows renters to explore average listing prices in specific neighborhoods as well as filter locations by room type and average rating. The vis also allows hosts to compare their listing to others using the aforementioned characteristics, as well as see the most popular words used in listing descriptions.
